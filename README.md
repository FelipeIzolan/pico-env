# pico8-setup

### installation:

- ```git clone https://github.com/FelipeIzolan/pico8-setup.git```
- set pico.exe path in makefile.
- write your game in src/main.lua!
- ```make debug```

### make:

- debug: 
    compile and run the cartridge.

- compile: 
    compile the cartridge.

- load: 
    load the cartridge.
