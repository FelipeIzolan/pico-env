# pico8-setup

### installation:

- ```git clone https://github.com/FelipeIzolan/pico8-setup.git```
- edit PICO8 in makefile.
- write your game script in main.lua!
- ```make debug```

### make:

- debug: 
    compile and load the cartridge.

- compile: 
    compile the cartridge.

- run: 
    load the cartridge.
